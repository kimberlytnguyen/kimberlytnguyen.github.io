ZALA Hair Extensions

Art Direction. Graphic Design. UI Design.

2019 - 2020

Social Media: Where it Came to Life

I collaborated with the Marketing Director to strengthen brand presence through Instagram Stories. The goal was to create eye-catching and premium content while using native Instagram features to stay current and on-trend. As an extension of this work, I also created ads across various platforms (Pinterest, Facebook, Instagram) to promote the brand further.


Visual Refinement

I collaborated closely with the Marketing Director to analyze and identify where ZALA visually fits in the hair extension market. I updated the brand look and feel to feel more premium and refined, with the most impactful change being web banners.

Improving Website User Interface

I collaborated with the web team to design and improve web features on the consumer-facing website. I worked directly with the Chief of Technology to brainstorm and create mock-ups to improve site flow, user-friendliness, and user experience across various features of the website. If you'd like to see some of those mock-ups, contact me here.
